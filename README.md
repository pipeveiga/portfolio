# Pipe Veiga | Portfolio

Portfolio personal estatico, listo para publicar con GitHub Pages.

## Archivos principales

- `index.html`: contenido y estructura.
- `styles.css`: diseño responsive.
- `script.js`: interacciones pequenas.
- `assets/hero.png`: imagen principal del hero.

## Publicacion en GitHub Pages

1. Ir a `Settings` en el repositorio.
2. Entrar en `Pages`.
3. Elegir `Deploy from a branch`.
4. Seleccionar `main` y carpeta `/root`.
5. Guardar.

El sitio queda publicado cuando GitHub termine el deploy.
